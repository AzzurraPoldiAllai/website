Il tuo sito web.

Crea e pubblica il tuo sito web in modo semplice e veloce utilizzando iWeb, Pages e molte altre applicazioni disponibili
per Mac OS X.

Dal tuo Mac, puoi creare e pubblicare il tuo sito web in un attimo. Quando il sito è pronto, potrai pubblicarlo con la stessa facilità.

Apri Preferenze di Sistema e fai clic su Condivisione, quindi seleziona Condivisione web.

Ecco fatto: il tuo sito adesso è disponibile sul tuo network privato, al lavoro o a casa.

Se sei connesso a Internet, allora il sito è disponibile anche per i tuoi amici, ovunque si trovino: per condividerlo con loro, invia loro l'indirizzo che viene mostrato nelle preferenze Condivisione.

Apache Power

La condivione web è basata sul server web Apache, una tecnologia standard del settore, inclusa in Mac OS X. Per ulteriori informazioni sul server web Apache, consulta il Manuale Apache.
